Описание плагина "Skard. Вывод случайного топика в тултипе"

Плагин добавляет в тулбар кнопку с иконкой игральной кости по нажатию на которую возникает тултип с произвольным топиком. Есть возможность настроить минимальный рейтинг. Плагин работает на Ajax

Лицензия:

Данный плагин распространяется по лицензии GNU GPL. Полный текст лицензии находится в файле LICENSE.txt
